# minigrep
rust practice from The Book

#### Simple `cargo` Commands
---

To run (w/o args): 
```
cd minigrep
cargo run
```

To run tests:
```
cargo test
```

#### Usage:
---

To use `minigrep`, you can run it a few ways.

Basic `cargo run`:
```
cargo run -- <your_search_string> file_to_search.txt
```
